exporting a face as a .dxf file

create a sketch, and double click on the face you want to save as dxf. it can only do one face at a time, so beware.

the 3d work we are using to as an example is a box, from here.

then, right click a sketch that is complete, and select "save as dxf". I imported it into inkscape, and placed them out evenly to save space and materials when I want to laser cut it.
