First, we draw a circle. then by using several offset planes, make more circles

Then, by using the loft function, we create a 3d model by connecting all the circles.

Of course, an egg holder must hold an egg, so we repeat our first step to make more circles

we repeat the loft function for those circles we drew to create a hole in the holder
