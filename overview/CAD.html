we are using the fusion 360 software to do 2d and 3d modelling

we will make an egg holder as an example.

NOTE: There is no absolute method to making something in these apps. As long as it is reasonable and applicable, it is a-ok.

First, we draw a circle. then by using several offset planes, make more circles

Then, by using the loft function, we create a 3d model by connecting all the circles.

Of course, an egg holder must hold an egg, so we repeat our first step to make more circles

we repeat the loft function for those circles we drew to create a hole in the holder

by importing an image, we will make a chess piece

then, we draw around the image by using lines and splines.

after that, we draw a line through the middle of the image, and revolve about that line

finally, we draw the part that does not need a revolve, and extrude accordingly.

parameter settings

This is where the parameter settings is. You will want to have it as an easily ascessible shortcut, so the 2nd image will show you how

This is its interface. You can add custom words that will point to certain numbers, i.e. Wid=5mm, len=60mm etc.

First, draw the bottom. Then extrude it.

now draw on any side, and extrude it too.

by using the offset plane, create the top part of the box

as seen in the 3rd image, the side has parts where it clashes with the top part. we can "combine" them to remove the extra conflicting parts.

repeat for the remaining sides, and you are done

exporting a face as a .dxf file

create a sketch, and double click on the face you want to save as dxf. it can only do one face at a time, so beware.

then, right click a sketch that is complete, and select "save as dxf". i imported it into inkscape.
